 -- NVDA 2026 Market Data Summary --
A real‑time analytics dashboard built in Python to explore NVIDIA (NVDA) stock performance using automated data pipelines and engineered metrics. The project pulls live market data, cleans and transforms it, and visualizes trends such as moving averages, volatility indicators, RSI, and volume patterns. Designed to demonstrate practical skills in API data ingestion, feature engineering, and dashboard creation for financial analytics, the dashboard provides a clear, interactive view of NVDA’s market behavior and supports quick decision‑making.
